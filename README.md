# Scalable Capital — AI Platform PM Test Assignment

**Author:** Pavel Andreyeuski
**Date:** February 2026

## Live Site

Open `index.html` in any browser, or visit the GitHub Pages URL after deployment.

## Structure

- **`index.html`** — Presentation covering the AI idea, AJTBD analysis, evaluation framework, guardrails, and roadmap
- **`demo.html`** — Interactive proof-of-concept with a scripted demo and live AI explore mode (Gemini 2.5 Flash)

## Quick Start

Both HTML files are self-contained (inline CSS/JS, no build step). Open `index.html` directly in a browser to begin.

The scripted demo in `demo.html` works fully offline. The Explore Mode requires an internet connection for the Gemini API.
